## 概述

Clarinet 是在 Stacks 区块链上构建、测试和部署智能合约的最快方法。它为您提供本地开发网络、REPL、测试框架和调试工具，让您能够自信地发布高质量的 Clarity 代码。

![单簧管概述](/images/tools/clarinet/overview.svg)

要使用人工智能探索Clarinet功能，请复制并粘贴 [llms.txt](/tools/clarinet/llms.txt) 输入到你选择的大语言模型中。

## 主要特点

* [**利用强大的命令行界面**](/tools/clarinet/cli-reference) - 创建新项目，使用 clarinet requirements 管理您的智能合约及其依赖项，并通过内置的 REPL 与您的代码进行交互。
* [**使用 SDK 编写单元测试**](/tools/clarinet/sdk-introduction) - 使用Clarinet SDK在熟悉的JS环境中编写单元测试并验证合约行为。
* [**运行私有区块链环境**](/tools/clarinet/local-blockchain-development) - 启动一个本地开发网络，包含节点、矿工和API，以便您可以测试和集成您的代码。
* [**VSCode 扩展**](/tools/clarinet/vscode-extension) - 代码检查工具、逐步调试器，有助于编写智能合约（自动补全、文档等）。

## 安装

<TerminalPicker storage="macOs">
  ```terminal !! macOS
  $ brew install clarinet
  ```

  ```terminal !! Windows
  $ winget install clarinet
  ```

  ```terminal !! Cargo
  $ sudo apt install build-essential pkg-config libssl-dev
  ```

  ```terminal !! Binary
  $ wget -nv https://github.com/hirosystems/clarinet/releases/download/v0.27.0/clarinet-linux-x64-glibc.tar.gz -O clarinet-linux-x64.tar.gz
  $ tar -xf clarinet-linux-x64.tar.gz
  $ chmod +x ./clarinet
  $ mv ./clarinet /usr/local/bin
  ```
</TerminalPicker>
